JS / CSS / Image 压缩工具集
===========
fisker lionkay@gmail.com

js 可以选择 google closure compiler 或 UglifyJS 压缩

css 使用 yuicompressor 压缩

图片 使用 pngout.exe 压缩


*js/css文件必须为utf-8编码*

*文件命名规则 file.dev.ext => file.ext / file2.ext => file2.min.ext*
